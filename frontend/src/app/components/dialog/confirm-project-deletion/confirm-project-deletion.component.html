Projekt löschen?

Wollen Sie Ihr Projekt wirklich unwiderruflich löschen? Dadurch werden alle mit dem Projekt verbundenen Daten unwiderruflich gelöscht:
  • alle verbundenen Mitglieder werden entfernt (lediglich die Verbindung wird aufgelöst)
  • alle Berichte werden entfernt
  • alle Meilensteine mit den verbundenen Prognosen werden entfernt
